# Stock Picker Web App

This is a simple stock picker web app that uses alphavantage APIs for fetching stock information.
Tech Stack: Javascript, React.js

## Features:
1. Search has auto-complete built-in.
2. The user can search for multiple stocks.
3. Last 5-day closing price is visualized on a graph using graph.js.
4. Stock information is auto-refreshed.

### Hosted Site: https://stock-picker.netlify.app
